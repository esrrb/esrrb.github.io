Transcriptome - mRNA Microarrays

Supplementary Data 1. Data from mRNA microarray experiments, using Affymetrix mouse gene 1.0 st array technology was collected using three biological replicates over days 0-1-3-5 after Esrrb knock-down. This data represents levels of gene expression across the genome as a function of time. Also included in the file is the log-transformed normalized data representing relative changes in gene expression relative to day 0. Download DTT File (347 MB)

Supplementary Data 2. Contains the list of significant genes differentially expressed after ANOVA ratio and 0.05 Bonferroni corrections in Esrrb knockdown time series experiment. Download XLSX file (6 MB)

All data have been deposited in the Gene Expression Omnibus (GEO) database with project identification number: GSE31842
